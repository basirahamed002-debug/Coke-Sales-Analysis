This project analyzes sales data for Coca-Cola products to identify trends, top-performing regions, and key growth drivers. The goal was to transform raw sales data into actionable insights to support decision-making strategies.

Key Objectives:

Analyze sales performance across different regions and time periods.

Compare operating profits and margins for various products.

Identify high-growth markets and underperforming segments.

Tools Used:

Google Sheets / Excel: For data cleaning, processing, and pivot tables.

Data Visualization: Charts and dashboards created to visualize trends.

Key Insights:

["Diet Coke sales spiked 20% in Q3."]

["The Northeast region had the highest operating margin."]
